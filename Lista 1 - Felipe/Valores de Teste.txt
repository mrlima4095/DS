### 4. Celsius → Fahrenheit
- **C = 0** → F = `(0 + 160)/5 = 32`
- **C = 100** → F = `(900 + 160)/5 = 212`
- **C = 25** → F = `(225 + 160)/5 = 77`

---

### 5. Fahrenheit → Celsius
- **F = 32** → C = `(32 - 32) * 5/9 = 0`
- **F = 212** → C = `180 * 5/9 = 100`
- **F = 77** → C = `45 * 5/9 = 25`
